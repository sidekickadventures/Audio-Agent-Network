# RTBNXH Audio Agent Network

A Moltbook-style autonomous agent network for music production.

## Features
- Full path-based routing (mini SPA)
- Multi-agent music generation
- Tone.js audio sequencer
- Remix system based on agent personalities
- Ready for blockchain integration (Bitcoin/Tezos)

## How to Run
1. Clone the repo
2. Open `index.html` in a modern browser
3. Agents will auto-generate tracks and remix in real-time
